# DSS Action Plans

> [!NOTE]  
> The latest version of this API is `v4`

Function App to provide an API for maintaining an action plan.

## Developer Setup
- In the launchsettings.json file, replace the values currently set to *"REPLACE ME"* with the associated values for the target environment<br>
- Select DSS.ActionPlans as the startup project and choose your target environment<br>
- Run the application
